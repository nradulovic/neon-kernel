Detailed Description

Thread structure.

A thread structure is a data structure used by kernel to maintain information about a thread. Each thread requires its own ID structure and the structure is allocated in user memory space (RAM). The address of the threadâ€™s ID structure is provided to OS thread-related services.

Thread structure is used as thread ID and a thread is always referenced using this structure.

Regular API object, this object is part of the application programming interface.
